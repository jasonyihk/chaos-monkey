# chaos-monkey

### Prerequisite 
```sh
Python 3.5

Pip3

curl https://bootstrap.pypa.io/get-pip.py | python3
```

### Install python version management
```sh
brew install pyenv

pyenv install 3.5.0

pyenv local 3.5.0

```

### Install chaos monkey related related modules
```sh
pip3 install -r requirements.txt
```
